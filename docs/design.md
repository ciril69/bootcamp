# design.md

# Design Vision

Create a premium, immersive streetwear shopping experience that feels bold, modern, and luxurious. The interface should emphasize the clothing through large imagery, clean layouts, subtle animations, and a consistent dark aesthetic.

---

# Brand Personality

* Premium
* Modern
* Minimal
* Urban Streetwear
* Bold
* Confident

The website should feel comparable to premium fashion brands rather than a traditional online store.

---

# Color Palette

### Primary Background

* Matte Black (#0B0B0B)
* Charcoal (#141414)

### Secondary Background

* Dark Gray (#1C1C1C)

### Accent

* **Shiny Metallic Gray**

  * Used for buttons
  * Borders
  * Icons
  * Hover effects
  * Card highlights
  * Navigation indicators

### Typography

Primary Text

* White (#FFFFFF)

Secondary Text

* Light Gray (#C8C8C8)

Muted Text

* Gray (#7E7E7E)

---

# Typography

Suggested fonts

Headings

* Bebas Neue
* Anton
* Oswald

Body

* Inter
* Manrope
* Poppins

Large bold headings should create a luxury streetwear feel while body text remains clean and highly readable.

---

# UI Style

* Rounded corners (12–16px)
* Glassmorphism only where subtle and purposeful
* Soft shadows
* Metallic hover highlights
* Smooth transitions (200–300ms)
* Large whitespace between sections
* Edge-to-edge hero imagery

---

# Animations

* Smooth page transitions
* Fade-up section reveals on scroll
* Image zoom on hover
* Button glow using metallic gray accents
* Cart slide-in animation
* Product cards lift slightly on hover
* Smooth mobile navigation drawer

Animations should feel polished, not distracting.

# Mobile Experience

The design should be mobile-first.

Key considerations:

* Sticky bottom "Add to Cart" button on product pages
* Large touch targets
* Swipe-friendly image galleries
* Compact filter drawer
* Bottom spacing for comfortable scrolling
* Fast-loading responsive images

---

# Accessibility

* High contrast between text and background
* Keyboard navigation support
* Visible focus indicators
* Semantic HTML structure
* Alt text for all product images
* Responsive layouts supporting modern mobile, tablet, and desktop devices
