# PRD.md

## Project Overview

Build a modern, dark-themed demo e-commerce website for a men's streetwear apparel brand. The website will showcase premium clothing collections and provide an engaging shopping experience while allowing customers to request product customization through direct contact with the store.

---

## Problem

Many small apparel brands struggle to present their products in a premium and engaging way online. This project aims to demonstrate how a modern fashion website can improve brand perception and customer engagement without requiring a complete e-commerce backend.

---

## Target Users

* Men aged **12–30**
* Streetwear and casual fashion enthusiasts
* Customers interested in oversized clothing and custom apparel

---

## Core Features (Ranked)

### 1. Product Catalog

* Browse oversized T-shirts, hoodies, tank tops, and jackets.
* Search products.
* Filter by category and size.

### 2. Product Detail Page

* High-quality image gallery.
* Product description.
* Available sizes.
* Price display.
* "Add to Cart" functionality.
* Customization information.

### 3. Request Customization

Customers can request:

* Custom prints
* Embroidery
* Bulk orders

The page provides direct **email** and **phone** contact options instead of an online customization tool.

### 4. Shopping Cart

* Add and remove products.
* Update quantities.
* View subtotal.
* Continue shopping or proceed to checkout.

### 5. Demo Checkout

A complete checkout experience without payment processing.

After submission, display a confirmation message explaining that this is a demonstration website.

### 6. Contact Page

* Email
* Phone number
* Store location (optional)
* Contact form (demo)

---

## Out of Scope (Version 1)

* User accounts
* Authentication
* Wishlist
* Reviews and ratings
* Live inventory management
* Payment gateway integration
* Admin dashboard
* Real order processing
* Shipping integration

---

## Success Criteria

The project is considered successful if users can:

* Easily browse all clothing categories.
* Find products using search and filters.
* View attractive product pages.
* Add products to a shopping cart.
* Complete a demo checkout flow.
* Understand how to request custom apparel.
* Enjoy a responsive experience on desktop, tablet, and mobile devices.
* Experience a premium brand aesthetic with a dark theme and shiny metallic gray accents.
